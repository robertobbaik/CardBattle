# CardBattle 카드 구현 규칙 최종안

## 1. 문서 목적

이 문서는 CardBattle의 카드별 전투 규칙을 정리한 최종 구현 기준 문서다.
모든 카드 클래스는 `BaseCard.cs`를 상속받아 구현하는 것을 전제로 한다.

본 문서의 목적은 다음과 같다.

- 카드별 공격 / 특수 능력 / 공개 효과를 명확히 정의한다.
- 반사 데미지 처리 기준을 통일한다.
- 힐, 보호막, 초과 체력, 공개 효과의 의미를 정리한다.
- 구현자가 이 문서만 보고 카드별 동작을 구현할 수 있도록 한다.

---

## 2. 공통 전투 규칙

### 2.1 기본 공격 규칙

별도 규칙이 없는 카드는 기본 공격을 사용한다.

```text
선택한 적 카드에게 자신의 현재 HP의 100% 피해를 준다.
공격한 카드는 공격 대상에게서 반사 데미지를 받는다.
```

### 2.2 반사 데미지 규칙

반사 데미지는 공격 대상 카드가 공격자에게 되돌려주는 피해다.

```text
기본 반사 데미지 = 공격 대상 카드의 현재 HP 기준 피해
```

카드별 규칙에 따라 반사 데미지를 받지 않거나, 120%로 받는 등의 변형이 있을 수 있다.

### 2.3 퍼센트 피해 기준

별도 설명이 없는 퍼센트 피해는 공격자의 현재 HP를 기준으로 계산한다.

```text
100% 피해 = 공격자의 현재 HP
80% 피해 = 공격자의 현재 HP * 0.8
50% 피해 = 공격자의 현재 HP * 0.5
30% 피해 = 공격자의 현재 HP * 0.3
120% 피해 = 기준 피해 * 1.2
```

소수점 처리는 프로젝트 구현 기준에 맞게 정하되, 우선은 `내림` 처리를 권장한다.
단, 계산 결과가 0보다 크고 1보다 작다면 최소 1 피해를 적용한다.

### 2.4 힐 규칙

힐은 HP를 회복시키는 효과다.

```text
힐은 최대 체력을 초과할 수 없다.
```

예시:

```text
MaxHP 10 / CurrentHP 9
힐 +2 적용
결과 CurrentHP 10
```

### 2.5 보호막 규칙

보호막은 HP와 동일하게 피해를 대신 받아내는 추가 체력이다.

```text
보호막은 최대 체력을 초과해서 HP를 증가시킬 수 있다.
```

예시:

```text
MaxHP 10 / CurrentHP 10
보호막 +2 적용
결과 CurrentHP 12
```

### 2.6 초과 체력 규칙

초과 체력은 현재 HP가 최대 HP를 넘어선 양을 의미한다.

```text
초과 체력 = CurrentHP - MaxHP
```

초과 체력이 0 이하라면 초과 체력이 없는 것으로 처리한다.

### 2.7 공개 효과 규칙

공개 효과는 카드가 대기 상태에서 전장에 배치되어 정보가 드러나는 순간 발동한다.

```text
카드 공개
↓
공개 효과 발동
```

사령관과 샤먼은 공개 효과를 가진다.

# 3. 카드별 최종 규칙 요약

| 카드 | 최종 규칙 요약 |
|---|---|
| 일반 카드 | 선택한 적에게 현재 HP 100% 피해를 주고, 반사 데미지를 받는다. |
| 원거리 카드 | 선택한 적에게 현재 HP 100% 피해를 주고, 반사 데미지를 받지 않는다. |
| 무쌍 카드 | 선택한 적에게 현재 HP 100% 피해를 주고, 인접한 적 카드 중 무작위 1장에게 현재 HP 50% 추가 피해를 준다. 추가 피해에 대해서는 반사 데미지를 받지 않는다. |
| 힐러 카드 | 매 턴 시작 시 아군을 1 회복한다. 힐은 최대 체력을 초과할 수 없다. 공격은 기본 공격과 동일하다. |
| 수호자 카드 | 공격 시 현재 HP 80% 피해를 준다. 수호자가 반사 데미지를 줄 때는 120% 피해를 준다. |
| 암살자 카드 | 공격으로 대상을 한 번에 처치하면 반사 데미지를 받지 않는다. 처치하지 못하면 반사 데미지를 120% 받는다. |
| 광전사 카드 | 공격 시 현재 HP 120% 피해를 준다. 공격 후 자신은 최대 HP 20% 피해를 받는다. 반사 데미지는 정상적으로 받는다. |
| 샤먼 카드 | 공개될 때 상대 공개 카드들의 초과 체력만큼 피해를 준다. |
| 사령관 카드 | 공개될 때 아군에게 보호막 2를 준다. 보호막은 최대 체력을 초과할 수 있다. |
| 폭탄병 카드 | 공격 시 상대 필드에 공개된 모든 카드에게 현재 HP 30% 피해를 준다. 반사 데미지를 받지 않는다. |

# 4. 카드별 상세 규칙

---

## 4.1 일반 카드

### 능력 설명

```text
선택한 적 카드에게 자신의 현재 HP의 100% 피해를 줍니다.
공격한 뒤 공격 대상에게서 반사 데미지를 받습니다.
```

### 처리 흐름

```text
일반 카드 공격
↓
선택한 적 카드에게 CurrentHP * 1.0 피해
↓
공격 대상에게서 반사 데미지 받음
↓
행동 종료
```

### 구현 메모

- 가장 기본이 되는 공격 카드다.
- 별도 특수 능력은 없다.
- 기본 공격과 반사 데미지 처리의 기준 카드로 사용한다.

---

## 4.2 원거리 카드

### 능력 설명

```text
선택한 적 카드에게 자신의 현재 HP의 100% 피해를 줍니다.
공격 후 반사 데미지를 받지 않습니다.
```

### 처리 흐름

```text
원거리 카드 공격
↓
선택한 적 카드에게 CurrentHP * 1.0 피해
↓
반사 데미지 처리 생략
↓
행동 종료
```

### 구현 메모

- 대상이 살아남아도 반사 데미지를 받지 않는다.
- 안정적으로 적을 공격하는 카드다.

---

## 4.3 무쌍 카드

### 능력 설명

```text
선택한 적 카드에게 자신의 현재 HP의 100% 피해를 줍니다.
추가로 선택한 카드와 인접한 적 카드 중 무작위 1장에게 자신의 현재 HP의 50% 피해를 줍니다.
인접한 적 카드가 없다면 추가 피해는 발생하지 않습니다.
```

### 반사 데미지 규칙

```text
선택한 공격 대상에게는 일반 공격과 동일하게 반사 데미지를 받습니다.
인접 카드에게 준 50% 추가 피해에 대해서는 반사 데미지를 받지 않습니다.
```

### 처리 흐름

```text
무쌍 카드 공격
↓
선택한 적 카드에게 CurrentHP * 1.0 피해
↓
선택한 공격 대상에게서 반사 데미지 받음
↓
선택한 카드의 인접 슬롯 확인
↓
인접한 적 카드 중 공개되어 있고 살아있는 카드 후보 생성
↓
후보가 있으면 무작위 1장 선택
↓
선택된 인접 카드에게 CurrentHP * 0.5 피해
↓
50% 추가 피해에 대한 반사 데미지는 없음
↓
행동 종료
```

### 구현 메모

- 인접 카드 후보가 없으면 추가 피해를 생략한다.
- 추가 피해 대상은 무작위 1장이다.
- 추가 피해는 공격 대상의 반격과 별개로 처리한다.

---

## 4.4 힐러 카드

### 능력 설명

```text
매 턴 시작마다 아군에게 힐을 합니다.
힐은 최대 체력을 넘어갈 수 없습니다.
공격은 기본 공격과 동일하게 처리합니다.
```

### 회복 수치

```text
아군 카드 HP +1
```

### 처리 흐름

```text
아군 턴 시작
↓
힐러가 전장에 공개되어 있고 살아있는지 확인
↓
아군 카드들에게 HP +1 적용
↓
각 카드의 HP는 MaxHP를 초과하지 않음
```

### 구현 메모

- 회복은 턴 시작 시 자동 발동한다.
- 회복은 행동을 소모하지 않는다.
- 힐러 자신을 회복 대상에 포함할지 여부는 현재 설계에서는 제외를 권장한다.
- 죽은 카드는 회복하지 않는다.

---

## 4.5 수호자 카드

### 능력 설명

```text
공격은 자신의 현재 HP의 80% 피해를 줍니다.
수호자가 반사 데미지를 줄 때는 120%의 피해를 줍니다.
```

### 처리 흐름 - 수호자가 공격할 때

```text
수호자 공격
↓
선택한 적 카드에게 CurrentHP * 0.8 피해
↓
공격 대상에게서 반사 데미지 받음
↓
행동 종료
```

### 처리 흐름 - 수호자가 반사 데미지를 줄 때

```text
상대 카드가 수호자를 공격
↓
수호자가 반사 데미지를 줌
↓
반사 데미지 = 기본 반사 데미지 * 1.2
```

### 구현 메모

- 수호자는 직접 공격 피해가 낮다.
- 대신 공격당했을 때 강한 반사 데미지를 준다.
- 방어형 카드로 취급한다.

---

## 4.6 암살자 카드

### 능력 설명

```text
공격을 했을 때 상대방의 체력을 한 번에 0으로 만들면 반사 데미지를 받지 않습니다.
한 번에 0으로 만들지 못한다면 반사 데미지를 120% 받습니다.
```

### 처리 흐름

```text
암살자 공격
↓
선택한 적 카드에게 CurrentHP * 1.0 피해
↓
대상 HP <= 0 인가?
  Yes → 반사 데미지 없음
  No  → 암살자가 반사 데미지 120% 받음
↓
행동 종료
```

### 구현 메모

- 처치에 성공하면 안전하다.
- 처치에 실패하면 큰 리스크를 받는다.
- 핵심 카드를 마무리하는 역할에 적합하다.

---

## 4.7 광전사 카드

### 능력 설명

```text
광전사는 공격 시 자신의 현재 HP의 120% 피해를 줍니다.
공격 후 자신은 최대 HP의 20%만큼 피해를 받습니다.
반사 데미지는 정상적으로 받습니다.
```

### 처리 흐름

```text
광전사 공격
↓
선택한 적 카드에게 CurrentHP * 1.2 피해
↓
공격 대상에게서 반사 데미지 받음
↓
광전사 자신에게 MaxHP * 0.2 피해
↓
행동 종료
```

### 구현 메모

- 광전사는 강한 피해를 주지만 스스로도 피해를 입는다.
- 반사 데미지는 정상적으로 받는다.
- 자해 피해는 공격 후 처리한다.
- 자해 피해로 광전사가 사망할 수 있다.

---

## 4.8 샤먼 카드

### 능력 설명

```text
카드가 공개될 때 상대방의 오픈되어 있는 카드 중 체력이 최대 체력을 넘어서는 만큼 데미지를 줍니다.
```

### 처리 흐름

```text
샤먼 공개
↓
상대 필드의 공개 카드 목록 확인
↓
각 카드별 초과 체력 계산
초과 체력 = CurrentHP - MaxHP
↓
초과 체력이 1 이상이면 해당 수치만큼 피해 적용
↓
초과 체력이 없으면 피해 없음
```

### 구현 메모

- 사령관의 보호막 효과를 카운터하는 카드다.
- 초과 체력만큼만 피해를 준다.
- CurrentHP가 MaxHP 이하인 카드에게는 피해를 주지 않는다.

---

## 4.9 사령관 카드

### 능력 설명

```text
카드가 공개될 때 아군에게 2의 보호막을 줍니다.
보호막은 HP와 같지만, 최대 체력 이상으로 HP를 늘려줄 수 있는 효과입니다.
```

### 처리 흐름

```text
사령관 공개
↓
아군 공개 카드 목록 확인
↓
각 아군 카드에게 보호막 +2 적용
↓
CurrentHP += 2
↓
MaxHP 초과 허용
```

### 구현 메모

- 보호막은 HP처럼 피해를 받는다.
- 보호막은 최대 체력을 초과할 수 있다.
- 샤먼은 이 초과 체력을 제거하는 방향으로 동작한다.

---

## 4.10 폭탄병 카드

### 능력 설명

```text
공격하면 상대방의 필드에 공개된 모든 카드에게 30%의 데미지를 줍니다.
반사 데미지를 받지 않습니다.
```

### 처리 흐름

```text
폭탄병 공격
↓
상대 필드의 공개 카드 목록 확인
↓
각 공개 카드에게 CurrentHP * 0.3 피해 적용
↓
반사 데미지 없음
↓
행동 종료
```

### 구현 메모

- 단일 타겟 공격이 아니라 상대 공개 카드 전체 공격이다.
- 대기 카드에는 피해를 주지 않는다.
- 공개되지 않은 카드는 피해 대상이 아니다.
- 반사 데미지를 받지 않는다.

# 6. 최종 요약

```text
일반 카드:
현재 HP 100% 피해, 반사 데미지 받음

원거리 카드:
현재 HP 100% 피해, 반사 데미지 없음

무쌍 카드:
현재 HP 100% 피해
인접 적 카드 중 무작위 1장에게 현재 HP 50% 추가 피해
기본 공격 대상에게는 반사 데미지 받음
50% 추가 피해에 대해서는 반사 데미지 없음

힐러 카드:
```
